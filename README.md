PulseFlow CRM 

Zero-Click Intelligence for East African Sales Teams

PulseFlow CRM eliminates manual data entry by automatically processing emails, calls, and meetings to keep your sales pipeline updated in real-time. Built specifically for Kenyan businesses with local context understanding.

 The Problem

East African sales teams waste 4.5 hours daily on manual CRM data entry, costing Kenyan businesses KES 2.5 billion annually in lost productivity. Traditional CRMs are desktop-heavy, USD-priced, and not built for mobile-first African markets.

 The Solution

PulseFlow CRM delivers true zero-click automation:

· Automatic contact extraction from emails
· Real-time sentiment analysis and deal scoring
· Natural language query interface
· Mobile-first design for Kenyan workforce
· KES-native pricing and currency handling

Tech Stack

Frontend:

· React 18
· Tailwind CSS
· Progressive Web App (PWA)

Backend:

· Node.js + Express
· PostgreSQL
· WebSockets for real-time updates

AI/Integrations:

· Custom NLP engine (94% accuracy)
· Gmail API integration
· Google Calendar API
· Kenyan context processing

 Quick Start

Prerequisites

· Node.js 16+
· PostgreSQL
· Gmail API credentials
